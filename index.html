View Project  

Android Studio

For develop this mobile application , used Android Studio with JAVA.

Firebase

Firstore Database service on Firebase used as database and fire storage used to upload images.

Google Maps

This project used google maps with nearby API.

Adobe XD

Adobe XD used for design front-end of this mobile application.

Features

Responsive Design

This app has eye catching design and no need to worry about how to use it.

App shows nearby places

This app shows to user their nearby places to buy products and services.

Support to the lowest and the latest versions

This application can install the lowest Android version API level 19 (Andriod 4.4 Kitkat version) to the latest API levels.

Supports multiple languages

This app support to English, Sinhala and Tamil Languages.

Educate and Motivate

App has a user guide manual, it runs at first installment, and users motivated via online videos.

Map View

User can see nearby products and services via map view with different colors.

Clean and Modern Design.

Divi Saviya app is an e-commerce application that is developed to hack: bit 2020 national level hackathon. This app helps people increase their per-capita income by becoming self-businessman. This platform helps people sell their products and services. The best thing is users can find products and services from nearby places. And also if user wants some product or service he/she can request it. Other users can see that requests and can fulfill it. And many more services can be found here

This application got the first place because of the applicable idea for COVID 19 pandemic period and for the good implementation. This app includes lots of technical features like autofill text fields, support multiple languages and so on.
